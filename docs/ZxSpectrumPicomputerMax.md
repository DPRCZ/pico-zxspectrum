### ZxSpectrumPicomputerMax
These is the target for Bobricius' Retro PICOmputerMAX.
<img src="picomputermax.png" height="300"/>

It supports the following:
* USB keyboard
* USB joysticks
* Matrix keyboard (6x6)
* LCD video (ST7789)
* PWM sound (1 pin)
* SPI SD card
