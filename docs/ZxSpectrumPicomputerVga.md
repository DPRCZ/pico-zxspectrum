

### ZxSpectrumPicomputerVga
This is the target for Bobricius' Retro VGA board:

<img src="retrovga.png" width="300"/>

It supports the following:
* USB keyboard
* USB joysticks
* Matrix keyboard (6x6)
* VGA video (RGB332)
* PWM sound (1 pin)
* SPI SD card
