### ZxSpectrumPicomputerZxLcd
This is a target written for Bobricius' PICOZX

<img src="picomputer_picozx.png" width="300"/>

It supports:
* USB keyboard
* Keyboard martix
* USB joysticks
* VGA video (RGB222)
* PWM sound (1 pin)
* SPI SD card

While on the menu...
* SHIFT-Fire = ESC
* RELOAD = Backspace
* SHIFT-RELOAD = Del
